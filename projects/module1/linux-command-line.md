## Key Commands Used

- Navigation: pwd, ls, cd
- File Operations: mkdir, cp, mv, rm, cat, nano, echo
- Permissions: chmod, ls -l
- Package Management: apt-get update, apt-get install
- Process Control: ps, htop, grep, background jobs
- User Management: adduser, groupadd, usermod, getent
- File Transfer: wget
- Archiving: tar
- System Info: date, whoami, hostname
- Redirection: >, >>, 2>
- Recording: asciinema rec/play
- Directory Tree: tree

## Key Lessons

- Precision in command syntax is critical
- File naming conventions must be followed exactly
- Linux permissions provide granular access control
- Background processes enable long-running tasks
- Output redirection separates standard output from errors
- Remote access requires proper authentication
- Documentation through session recording aids learning
